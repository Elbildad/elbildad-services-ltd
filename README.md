# Elbildad Services LTD - China-Nigeria Sourcing & Procurement

Official website for **Elbildad Services LTD**, a fast-growing China-to-Nigeria procurement, sourcing, and business development company.

## Features
- **Sourcing & Procurement:** Direct access to Chinese manufacturers.
- **Supplier Vetting:** Quality assurance and factory audits.
- **Cross-Border Payments:** Secure CNY/USD/HKD payments.
- **RFQ System:** Streamlined Request for Quotation process with commitment fee integration.
- **Responsive Design:** Modern, mobile-first design with dark mode support.

## Tech Stack
- **Framework:** Next.js 15
- **Library:** React 19
- **Styling:** Tailwind CSS
- **Icons:** Iconify
- **Animations:** AOS (Animate On Scroll)

## Getting Started

### Prerequisites
- Node.js (v24 or later recommended)
- npm or pnpm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Production Build
```bash
npm run build
npm run start
```
